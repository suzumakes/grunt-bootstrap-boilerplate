Grunt

Isn't this the whole point? Grunt runs your tasks. Whatever you need to do to prepare your site/app/whatever for production, Grunt does it all with a simple command and a little spinning "/". This boilerplate has Grunt process SCSS to CSS, concatenate and mangle JavaScript, Copy and Compress images to their correct directories, minify CSS, and build HTML pages using Includes -- all with one command: "grunt".

Sass

Sass is CSS with Superpowers, made by this awesome gentleman here providing support for nesting, functions, variables, and mixins. In fact, every CSS file is already valid SCSS! Add your _scss files, include them in style.scss, and Grunt will process them to CSS for you.

Reminder: It's Sass, not SAAAASSSS!

Bootstrap

Twitter is fantastic, and Twitter Bootstrap is one of the most complete responsive front-end frameworks available, and with a Sass distribution available and version 4 on the way with greater flexibility with fewer nested classes, it's only getting more relevent and useful.
